# 🛒 Lista de Compras - PWA

Aplicação web progressiva (PWA) para gerenciamento de lista de compras, desenvolvida com:

- HTML5  
- CSS3  
- JavaScript Vanilla  
- Service Worker  
- LocalStorage  

🔗 Acesse o projeto online:  
https://lucasfreire99.github.io/compras-app/

---

## 📱 Funcionalidades

- ✅ Adicionar itens
- ✏️ Editar itens
- 🗑 Excluir itens
- 🧮 Cálculo automático (quantidade × valor)
- 💰 Total geral em tempo real
- 🗂 Organização por categoria
- 🔎 Filtro por categoria
- 📋 Copiar lista formatada
- 💾 Salvamento automático no navegador
- 📲 Instalável como aplicativo (PWA)
- 🌐 Funciona offline

## 🚀 Como Instalar

1. Acesse o link do projeto.
2. No navegador compatível (Chrome/Edge), clique em **"Instalar App"**.
3. O aplicativo será adicionado à tela inicial do dispositivo.
